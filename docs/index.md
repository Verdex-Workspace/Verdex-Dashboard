## En bref

Le **Verdex Dashboard** est une application **Vue 3 + TypeScript** construite sur
un design system maison. Elle centralise le pilotage de vos projets et de votre
infrastructure pour vous (et vos clients) dans une seule interface.

- 🚀 **Pour démarrer côté technique** → [Prise en main](/guide/getting-started)
- 🧑‍💼 **Pour utiliser le dashboard au quotidien** → [Guide utilisateur](/user-guide)
- 🛠️ **Pour brancher votre infra (Grafana, Proton, n8n…)** → [Checklist infrastructure](/infra-setup)
